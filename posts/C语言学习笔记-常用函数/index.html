Scanf函数

C函数库包含了多个输入函数，scanf是最通用的一个，可以读取不同格式的数据

基本用法

VS中scanf()会被警告不安全，需要用scanf_s代替scanf

#include <stdio.h>
void main()
{
	int num1,num2;
	printf("Please input a number1:");
	scanf_s("%d", &num1);
	fflush(stdin);//清空缓冲区
	printf("The num is: %d\n",num1);
	printf("Please input a number1:");
	scanf_s("%d", &num2);
	printf("The num is: %d\n", num2);
	char ch;
	printf("Please input char: ");
	ch = getchar();
	//scanf_s("%c", &ch);
	printf("Input char is: %c\n",ch);
}

运行结果：

Please input a number1:1
The num is: 1
Please input a number1:2
The num is: 2
Please input char: Input char is:

从结果上看，字符串没有输出(其实可以连续输入达到输出字符串)，C处理字符比较复杂，这个问题以后再讲。

转义序列

转换字符串 含义
%d 把输入解释成有符号整型
%c 把输入解释成字符型
%s 把输入解释成字符串
%f 把输入解释成单精度浮点数
%lf 把输入解释成双精度浮点数
